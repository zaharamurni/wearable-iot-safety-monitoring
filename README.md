# Wearable IoT-Based Safety Monitoring System

## Overview
This project was developed as part of my bachelor thesis in Telecommunications Systems at Universitas Pendidikan Indonesia.

The system is designed to improve safety in public transportation environments through real-time proximity monitoring using wearable IoT technology.

## Problem Statement
Public transportation users may face safety risks due to limited awareness of surrounding objects and distances. This project aims to provide real-time monitoring and alerts to enhance user safety.

## Technologies Used
- ESP32
- Time of Flight (ToF) Sensor
- Blynk IoT Platform
- Arduino IDE

## Features
- Real-time distance detection
- Vibration alert system
- Mobile monitoring through Blynk
- IoT-based data communication

## Testing
The system was evaluated through:
- Sensor accuracy testing
- Response time testing
- False alarm testing
- Environmental testing
